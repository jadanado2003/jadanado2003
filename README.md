## Featured projects

### HelpDeskPro — IT Helpdesk Ticketing and Asset Management System

A full-stack IT helpdesk system for managing support tickets, users, comments, internal notes, priorities, statuses, technician assignment, and IT assets.

This project is designed to simulate a real-world internal IT support platform used by service desk teams to organise support requests and manage company devices.

**Tech used:** React, TypeScript, Node.js, Express, Prisma, PostgreSQL, JWT authentication, CSS

Key features:
- User authentication with JWT
- Role-based access control
- Ticket creation and management
- Ticket priorities and statuses
- Technician assignment
- Ticket comments and internal notes
- Asset and device management
- Dashboard statistics
- Backend API with Express
- Database modelling with Prisma and PostgreSQL
- Structured full-stack project layout

Repository: https://github.com/jadanado2003/HelpDeskPro
